# demo-ios-login
用Wilddog方式，QQ登录、微信登录、微博登录、匿名登录demo

    Login 步骤
    第一步：打开www.wilddog.com ———— 登录 ———— 创建您的app
    第二步：管理应用 ———— 终端用户认证 ———— 社交账号登录 ———— 填写相关信息
    第三步：在xcode集成您需要的第三方登录SDK（QQ、weixin、weibo）
    第四步：在第三方登录成功的代理方法中，填写Wilddog的oAuth方法登录


## 克隆 demo
```
git clone git@github.com:WildDogTeam/demo-ios-wildchat.git
cd  demo-ios-wildchat
```
